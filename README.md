# 译者注

## 为什么创建这个项目

在我备考研究生考试时，发现英语语感尽失。

创建本项目来翻译这本书，是用来帮助我训练英语阅读技巧，找回英语语感。

同时，本翻译也基于兴趣与专业考量。数字信号处理是译者学术生涯中较为重要的一个内容，但本科阶段并未经过系统训练。译者意图借用此次翻译来作为数字信号的入门。

本人的翻译水平无法达到专业翻译水平，如发现谬误，望读者不吝赐教。
